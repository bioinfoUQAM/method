ATTRIBUTES

										<FEATURE> : Annotation feature 			eg. category
										<RANGE from="" to=""/> :contexte		eg. gather 2 previous attributes and 2 next atributes too
ATTRIBUTE								<NAME>Class</NAME> : class attribute
										<POSITION>0</POSITION> : This is an ATTRIBUTE, not an ATTRIBUTELIST, so we have "position", not "range"
